# case-studies

**Estudos de Caso Aplicados**

Onde a teoria encontra a realidade. Contem exemplos concretos de como o SOP-IA foi
aplicado em projetos reais, com os artefatos preenchidos como referencia.

## Para que serve
Eliminar a distancia entre "entender o framework" e "saber como usar". Ver um artefato
preenchido para um projeto real e mais eficaz do que ler sua descricao teorica.

## Quando consultar
- Quando tiver duvida de como preencher qualquer artefato
- Para onboarding de novos membros do time
- Para calibrar estimativas com base em projetos anteriores similares
